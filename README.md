# Learning Spring Security 🔐

This repository contains my learning journey and practice implementations of **Spring Security** while building secure backend applications using **Java and Spring Boot**.

The goal of this project is to understand how authentication, authorization, and security configurations work in real-world Spring Boot applications.

---

## 📚 Topics Covered

- Introduction to Spring Security  
- Authentication vs Authorization  
- In-Memory Authentication  
- Password Encoding (BCrypt)  
- Role Based Access Control  
- Security Configuration  
- HTTP Security Configuration  
- H2 Database Integration  
- Custom User Details Service  
- Login and Logout Handling  

---

## 🛠 Tech Stack

- Java
- Spring Boot
- Spring Security
- Maven
- H2 Database
- IntelliJ IDEA
- Postman
